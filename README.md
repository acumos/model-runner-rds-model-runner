# Acumos RDS Predictor Service

This repository holds the server components of the RDS (R Data) Predictor Service
for the Acumos machine-learning platform.  The service provides scoring mechanisms
for RDS based models which are models saved in R's RDS format.

Please see the documentation in the "docs" folder.
